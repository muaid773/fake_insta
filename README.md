# fake_insta

A simple project that simulates an Instagram login test page for educational and UI demonstration purposes only.

## Project Description

This project contains basic HTML files that display a fake Instagram-style login interface. It does **not** collect or process any real user data. The goal is purely to demonstrate UI structure.

## Project Structure

* `templates/instgram.html` – Main mock login page
* `templates/surce.html` – Additional page
* `templates/whaite.html` – Loading/transition page

## Notice

This project is for **educational and interface testing purposes only**.
